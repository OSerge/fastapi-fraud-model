Добро пожаловать!

Это API для предсказания мошеннических транзакций. За предсказания отвечает модель RandomForestClassifier.

Вы можете использовать Streamlit App для удобной загрузки файлов с датасетами,

а также OpenAPI (Swagger UI) как для загрузки файлов, так и для изучения API.

В консольном режиме используйте /predict для отправки CSV-файла с транзакциями.

Пример запроса:

curl -X POST "http://localhost:8000/predict" -F "file=@your_file.csv"

Пример ответа:

{
    "predictions": [
        {
            "id": 1,
            "is_fraud": false,
            "fraud_probability": 0.01,
            "fraud_risk": "low"
        },
        {
            "id": 2,
            "is_fraud": true,
            "fraud_probability": 0.85,
            "fraud_risk": "high"
        }
    ]
}
